1- LCD Without potentiometer

LCD Without potentiometer is a fairly simple project that you can build. 

 Components:

LCD 1x
Male to male jumper wires 11x
BreadBoard 1x
Arduino Uno 1x

Circuit Diagram:

Code: 
#include <LiquidCrystal.h>
int Contrast=75;
 LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

void setup()  { 
analogWrite(6,Contrast); lcd.begin(16, 2);  
}

void loop()  
{  
     lcd.setCursor(0.0); lcd.print("Khan Software");
     lcd.setCursor(0, 1); lcd.print("Hello World!");
}
